Spécification Officielle — Ether v0.3
1. Registres Disponibles
ACC (Accumulateur) : Stocke le résultat principal et la valeur de retour finale.

VAL (Valeur) : Registre secondaire pour les opérations à deux arguments.

CNT (Compteur) : Dédié à la gestion des boucles et indexations.

TMP (Temporaire) : Registre de travail rapide / jetable.

2. Jeu d'Instructions (Opcodes)
Mémoire & Registres
MOV dest, src : Copie la valeur/registre src dans dest.

STORE [adresse], reg : Écrit la valeur de reg dans la RAM.

LOAD reg, [adresse] : Lit la valeur de la RAM vers reg.

Arithmétique
ADD dest, src : Additionne (dest = dest + src).

SUB dest, src : Soustrait (dest = dest - src).

MUL dest, src : Multiplie (dest = dest * src).

INC reg : Incrémente de 1 (reg++).

DEC reg : Décrémente de 1 (reg--).

Contrôle de Flux (Nouveau v0.3)
nom_label: : Définit un point de repère dans le code.

JMP label : Saut inconditionnel vers le label.

CMP reg, val/reg : Compare deux valeurs et met à jour les drapeaux d'état (flags).

JE label : Saute au label si égalité (==).

JNE label : Saute au label si inégalité (!=)

Exemple complet en v0.3:
Compteur de 0 à 100 par pas de 25 

MOV CNT, 0         ; Initialise le compteur
MOV ACC, 0         ; Initialise le résultat

boucle_principale:
ADD ACC, 25        ; Ajoute 25 à ACC
INC CNT            ; Incrémente le compteur de 1

CMP CNT, 4         ; Vérifie si CNT == 4
JNE boucle_principale ; Si non, on boucle !

; Fin du programme : ACC vaut 100 !
